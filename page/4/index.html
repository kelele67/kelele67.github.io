Core Animation(目前苹果的API Reference已更名为Core Graphics)

Core Animation是iOS与OS X平台上负责图形渲染与动画的基础设施。Core Animation可以动画视图和其他的可视元素，为你完成了动画所需的大部分绘帧工作。你只需配置少量的动画参数（如开始点位置和结束点的位置）即可施展Core Animation魔法。Core Animation将大部分实际的绘图任务交给了图形硬件来处理，图形硬件会加速图形渲染的速度。这种自动化的图形加速技术让动画拥有更高的帧率并且更加的平滑，而且不会加重CPU的负担而影响程序的运行速度。

数据结构总结

  1. List
  2. Dictionary
  3. Stack

  5. Deque(双向队列)

List

list的append(value)和insert(0, value)，即一个从后面插入(后插)，另一个从前面插入(前插)，前者的效率远远高于后者。

检查括号匹配问题以及图的深度搜索…

Queue

需要优先队列的问题或者进行广度优先搜索的问题…

Deque

左右两边都可以插入和删除的队列

Binary_tree

一个节点最多有两个孩子节点的树。如果是从0索引开始存储，那么对应于节点p的孩子节点是2p+1和2p+2两个节点，相反，节点p的父亲节点是(p-1)/2位置上的点

算术表达式建立一颗二叉树可以清楚看出表达式是如何计算的
二叉树的三种遍历方法(前序，中序，后序)…

BinHeap

二叉堆：根据堆的性质又可以分为最小堆和最大堆，是一种非常好的优先队列。在最小堆中孩子节点一定大于等于其父亲节点，最大堆反之。二叉堆实际上一棵完全二叉树，并且满足堆的性质。对于插入和查找操作的时间复杂度度都是O(logn)O(logn)。

见到你真高兴

2016-10-01

从今天起，你要做一个不动声色的大人了。不准情绪化，不准偷偷想念，不准回头看。去过自己另外的生活。你要听话，不是所有鱼都会生活在同一片海里。
曾经那些美好都是早已经搁浅死在沙滩的鲸鱼。


小熊第一次看见正在吃草的兔子的时候，他的心像夏日里一瓷碗的酸梅汤，碎冰碰壁哐当当啷的响。

从那天起，小熊开始注意到这只每天都来自己家门前草地吃草的灰色小兔子。

兔子每天都起得很早，在太阳刚露出白色肚皮的时候，小熊就能看见她在沾满露水的草地上吃草，打滚。于是小熊想到了自己的小时候，有着妈妈的照顾，自己也是如此的无忧无虑，每天在吃喝玩乐中度过。记忆中的森林也是和现在呈现出一副不一样的光景，森林里每一棵树都像是一座碑塔，高耸入天，甚有气势。可是记忆不过是一种虚无缥缈的存在，它的存在仅仅是为了弥补时间的缺憾，好让我们在之后的漫长岁月里，多一个念想。

兔子吃草很快，太阳抿嘴一笑的功夫，刷刷刷的就能把一大块区域的草地全部啃光。

等到太阳再大一些的时候，兔子便会离开草地，一蹦一跳的融入周围的灌木丛。

日子就这样一天一天的过去，小熊总算是吃完了去年掏的一整罐蜂蜜，门口草地上的草却每天都一个样。

渐渐地，小熊忘记自己第一次见到兔子是什么时候了，他也从来没有和她说过话，甚至都没有从她的身边经过过。直到有一天，小熊发现兔子变了，她的毛发不再是黯淡无光转身就能融入灌木丛的灰色了，兔子披上了雪白的外衣，时不时还在阳光下露出漫长等待后的一丝闪光。

凛冬已至。

地球用了6亿年才诞生生命，然而小熊只用了一个冬眠的时间就决定鼓起勇气和兔子打招呼了。

春天早早的就到来了，期待中的兔子却迟迟没有出现。小熊并没有放弃，就像相信阿基里斯一定能追上乌龟一般相信兔子一定会再来自己家门前的这块肥美草地吃草的。

这段时间里，小熊每天都把自己安排的井井有条。

☑ : 早起跑步
☑ : 去河边钓鱼
☑ : 看望妈妈并带上一罐蜂蜜
☐ : 睡觉前看完一本书

就在春天的尾巴即将消失的时候，期待已久的画面终于到来了。

这天和往常一样，小熊早早的起床准备一天的行程，但是当他看见草地上那个熟悉的灰色身影的时候，心里先是咯噔了一下，然后终于鼓足了勇气，向兔子问了声好。

“你好哇兔子小姐”

“HelloWorld!小熊先生”
